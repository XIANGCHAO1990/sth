説明：スマートBMSアクセサリには、NTC温度制御（従来の構成）、Bluetoothモジュール、USB-UARTケーブル、USB-RS485ツールケーブル、CANbusモジュール、UART / RS485タッチLCDディスプレイ画面が含まれます。1. UART / RS485タッチLCDディスプレイ画面：関連するアイコンまたはロゴをカスタマイズでき、485 / UART通信をサポートし、一部のパラメーターのみを変更できます。2.Bluetoothモジュール：携帯電話に正しいBluetooth APP（SMART BMS）をダウンロードし、BluetoothモジュールをBMSの一致ポートに挿入します（挿入を間違えると破損します）。APPを開き、相対的なBluetoothのシリアル番号を確認すると、データを取得できます。3.電源表示パネル（SOCインジケーター） ：ボタンを押すことでBMSを起動する機能付き；ボード上に5つのライト、4つの黄色のライト、1つの赤色のライトがあり、各黄色のライトは25％の電力を表します。赤色のライトは、常に点灯している電源表示パネルのステータスライトを表します。SOCインジケータは、4ストリング以上（4ストリングを除く）のSMARTBMSとのみ互換性があります。4. USB-UARTケーブル： 3Pと6Pの2種類があります。 4ストリング未満（4ストリングを含む）または60A未満（60Aを含む）のBMS用の3P UARTケーブル、およびその他の場合の6PUARTケーブル。5. USB-RS485ツールケーブル： 保護ボードを焼き尽くさないように、通信インターフェースがランダムに接続されていないことに注意してください。6. CANbusモジュール： PCとBMS間のデータ送信ツールは、ドライブが正常にインストールされ、ラインの順序が正しいことを確認して、当社が提供する正常なグリブと通信する必要があります。7. NTCケーブル： 温度が上昇すると、抵抗は指数関数的に減少します。上位コンピューターとの485ボーレート通信：9600 / kbsまたは115200 / kbs上位コンピューターとのCANボーレート通信：250 / kbsまたは500 / kbs注意：1. BMSタッチコントロール画面とBluetoothモジュールを同時に使用することはできません。2つの同じインターフェイスのいずれかを選択してください。2. BMSに適したUSB-UARTケーブルのタイプを選択するように注意してください：3Pと6Pの2つのタイプがあります。 4ストリング未満（4ストリングを含む）または60A未満（60Aを含む）のBMS用の3P UARTケーブル、およびその他の場合の6PUARTケーブル。3. SOCインジケーターは、4弦以上（4弦を除く）のSMARTBMSとのみ互換性があります。パッケージは以下を含みます：1 x USB-UARTケーブル3P（オプション）1 x USB-UARTケーブル6P（オプション）1 x USB-RS485ケーブル（オプション）1 xパワーディスプレイパネル（オプション）1 x Bluetoothモジュール（オプション）1 x UARTタッチコントロール画面（オプション）1 x RS485タッチコントロール画面（オプション）1 x USB to CANボックス（オプション）1 x 1 x NTCケーブル（オプション）
