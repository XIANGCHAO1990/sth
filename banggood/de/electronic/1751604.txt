Danke.Beschreibung:Die Wetterstation verwendet den ESP8266-12E, um Daten aus dem Internet abzurufen: Uhrzeit einer Stadt, Wetterdaten und Prognoseinformationen für die nächsten 3 Tage, Bildlauf auf dem SSD1306 OLED-Display; Das Gerät kann so umschalten, dass Daten aus jeder Stadt der Welt angezeigt werden - vielleicht leben Ihre Verwandten oder Freunde dortDas Gerät verwendet die Sensoren DHT11, BMP180, BH1750FVI, um Temperatur-, Feuchtigkeits-, Atmosphärendruck- und Lichtdaten zu erfassen. Die Wetterstation liest alle 5 Sekunden Daten in Innenräumen über einen Sensor und lädt sie alle 60 Sekunden ins Internet hoch. Sie können Echtzeit-Datendiagramme von Ihrem Telefon oder Computer aus anzeigen. Natürlich können Sie den Code ändern, um verschiedene Funktionen zu implementierenESP8266: CP2102 Chip im Inneren, Micro-USB, 4 MB Flash, voller E / A-Anschluss und drahtloses 802.11, unterstützt von der IOT-Plattform.DHT11-Sensor: Ein digitaler Signalausgang mit einem kalibrierten kombinierten Temperatur- und Feuchtigkeitssensor. Das einfache Kommunikationsprotokoll reduziert den Programmieraufwand erheblichBMP180: hochpräziser 3-A-Drucksensor mit geringer Größe und extrem geringer Leistung. Durch seine hervorragende Leistung kann die absolute Genauigkeit des niedrigsten 0,03 hPa erreichen. Kann direkt über den I2C-Bus erreicht werden, der mit einer Vielzahl von Mikroprozessoren verbunden ist. Druckbereich: 300 ~ 1100hPa.BH1750FVI: Digitaler Lichtintensitätssensor. Direkter digitaler Ausgang, komplexe Berechnung entfällt, Kalibrierung entfällt.OLED Gelbblau-Display: 0,96 "SSD1306 128X64 Gelb & Blau OLED LCD Display mit I2C IIC SPI Serial Ergebnis:1) Holen Sie sich in regelmäßigen Abständen Wetterdaten von OpenWeathermap, die das Wetter von heute und die Wettervorhersage für die nächsten 3 Tage in jeder Stadt der Welt anzeigen2) Lesen Sie in regelmäßigen Abständen die aktuellen Temperatur- und Feuchtigkeitsdaten im Innenbereich3) Regelmäßiges Ablesen des Luftdrucks, Lichtintensitätsdaten4) Laden Sie in regelmäßigen Abständen Temperatur, Luftfeuchtigkeit, Luftdruck und Intensität der Beleuchtungsdaten auf thingspeak.com hoch5) Sehen Sie sich die Wettervorhersage auf dem OLED-Display an und sehen Sie sich die Umgebungsüberwachungskarte auf thingspeak.com anFührungsschritt:Übersicht: In diesem Handbuch wird beschrieben, wie eine Miniaturwetterstation per IDE erstellt wirdSchritt 1: Installieren Sie USB-To-SerialSchritt 2: Installieren Sie die IDE für ESP8266Schritt 3: Brennen der Firmware auf ESP8266Schritt 4: Komponenten anschließenSchritt 5: Registrieren Sie OpenWeathermap, ein neues KontoSchritt 6: Importieren Sie den WeatherStation-Code in die IDESchritt 7: Bibliothek hinzufügenSchritt 8: Ändern Sie die Datei WeatherStation.inoSchritt 9: Stellen Sie die Karte und den Port erneut einSchritt 10: Brennen Sie den Code auf ESP8266Schritt 11: ErgebnisLieferinhalt:1 x ESP8266-12E1 x DHT111 x BMP1801 x BH1750FVI1 x OLED-Display1 x USB-Kabel2 x Steckbrett20 x Dupont-Kabel
