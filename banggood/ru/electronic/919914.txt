Особенности:

Модуль (MicroSD адаптер карты) представляет собой модуль считывания карт Micro SD и интерфейс SPI через драйвер файловой системы, систему микроконтроллеров для завершения чтения и записи файлов MicroSD.
Arduino пользователи могут напрямую использовать Arduino IDE поставляется с SD-картой для завершения инициализации библиотечной карты и чтения-записи

Возможности модуля следующие:

Поддержка Micro SD Card (<= 2G), Micro SDHC-карта (<= 32G) (высокоскоростная карта)
Плата преобразования уровня, которая может поддерживать уровень интерфейса, составляет 5V или 3,3 В
Источник питания - 4.5V ~ 5.5V, 3.3V плата регулятора напряжения
Интерфейс связи - это стандартный интерфейс SPI
4 M2 отверстия для позиционирования отверстий для легкой установки
Размер: 4.1 x 2.4см

Интерфейс управления:
В общей сложности шесть выводов (GND, VCC, MISO, MOSI, SCK, CS), GND на землю, VCC - это блок питания, MISO, MOSI, SCK - шина SPI, CS - это пиковый сигнал выбора микросхемы

Цепь регулятора 3.3V:
Выход регулятора LDO 3.3V в качестве чипа преобразователя уровня, питания Micro SD

Схема преобразования уровня:
Micro SD-карта в направлении сигналов на 3,3 В, карта MicroSD в направлении интерфейса управления MISO-сигнал также преобразуется в 3,3 В, общая система микроконтроллера AVR может считывать сигнал

Разъем Micro SD:
Это держатель для самозарядки карт для удобной установки и снятия карты.

Позиционные отверстия:
4 M2 винты с отверстием в отверстии диаметром 2,2 мм, модуль прост в установке, для достижения межмодульной комбинации
