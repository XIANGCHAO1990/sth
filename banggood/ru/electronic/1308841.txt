Обзор:

Sonoff S26 - это умная розетка Wi-Fi с США / Великобритания / Австралия / DE / Франция / BR / CH / IL / IT стандарт. Пользователи могут мгновенно превратить любую вилку в интеллектуальную розетку с помощью удобной функции таймера, который работает на смартфоне пользователя. С помощью этой умной розетки вы можете включать / выключать любое подключенное устройство либо через приложение eWeLink, либо нажав ручку ручного управления на вилке WiFi. Это позволяет вам установить таймеры обратного отсчета / расписания / цикла для автоматического включения / выключения розетки, управляемой Wi-Fi, в приложении. Вы можете поделиться розеткой Wi-Fi со своей семьей, чтобы вы могли управлять вместе. Просто подключите и потратьте минуту на настройку, вы сможете управлять своим освещением и бытовой техникой из любого места в любое время. 

Функции:

Удаленное включение / выключение подключенного устройства
Поддержка проверки статуса устройства в реальном времени в приложении
Поддержка задач по расписанию / обратному отсчету / циклу
Настройка сцены для включения / выключения группы устройств в одной учетной записи
Настройка сцены для включения / выключения другого устройства при изменении статуса
Работает с Alexa
Работает с Google Home, Google Home Mini
Работает с IFTTT
