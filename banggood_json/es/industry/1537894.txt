Descripción:



La base magnética tiene un imán permanente dentro del cuerpo del bloque de metal, con el interruptor de "encendido / apagado" o "+ / -" para abrir o sombrear el campo magnético para sostener la superficie de metal ferroso o quitar la base magnética. La base magnética tiene un orificio para el tornillo en la superficie superior para instalar el soporte para el indicador de cuadrante, indicador, brazo de corte de línea, barra de elevación, etc.



Nota: 1, la fuerza de sujeción magnética depende del material ferroso de sujeción, el grosor, la planitud, la rugosidad, etc.



          2, 30 kg es el tipo de fuerza máxima de rotura de sujeción probada, no la fuerza de flexión o fricción.



          3, debido a que el imán interior está hecho de material Ndfeb fuerte, el soporte giratorio es especial, así que opere el interruptor
