Cómo utilizar:

1. Saque el enchufe USB.

2. Enchufe el cargador extraíble.

3. Encienda el cargador extraíble.

4.Presione el interruptor de control de temperatura del pecho durante dos segundos.

5. Luz roja intermitente, significa calentamiento automático.

6. Después de que la luz roja parpadee durante 5 minutos, luego el cambio automático a luz blanca, significa temperatura media.

7. El ajuste manual tiene tres niveles, la luz roja significa calentamiento continuo a alta temperatura.

La luz blanca significa calentamiento medio continuo, la luz azul significa calentamiento bajo continuamente.

8.Presione el interruptor de control de temperatura del pecho durante dos segundos, apague las luces y deje de calentar.



Instrucciones de lavado:

1. Saque el cargador extraíble.

2. El detergente neutro está disponible y luego se lava a mano.

3. No fuerce amasar o batir.

4.Después de la naturaleza, se puede usar el secado al aire.



Consejos cálidos:

1. Si no se usa durante mucho tiempo, no conecte el cargador extraíble

2. No use agua a alta temperatura ni detergente ácido fuerte.

3. Prohibido usar electricidad para ropa mojada, apague y deje de usar si la tela está mojada.

4. Tiempo de uso según la capacidad de carga extraíble.
