Descripción:

1. Este kit tiene una función de buscador de automóviles que hará parpadear las luces de su vehículo.

2. ¡Agregue un relevo y también tocará la bocina! Si tiene un motor para abrir su baúl, este kit le permitirá abrir su baúl con su control remoto.

3. Finalmente, si tiene un módulo enrollable de ventana o cualquier otro accesorio que le gustaría operar, este kit le permite controlar hasta 2 dispositivos opcionales con el control remoto.

Seguridad:

1. Matanza inicial

    Desactiva el vehículo para que se inicie mientras la alarma está armada.

2. Stage Duel Shock Sensor

    Los impactos bajos hacen que la sirena emita sonidos de advertencia mientras que los impactos más grandes hacen sonar la alarma.

3. Dome Retardo de disparo de luz

    Cuando está habilitado, el sistema de alarma no armará por completo la alarma hasta que todas las puertas estén cerradas.

4. Disparador de encendido

    Cuando el sistema de alarma está activado, la alarma sonará si el encendido recibe alimentación o si se manipula.

5. Pasador de campana y / o Trunk Trigger

    Cuando el sistema de alarma está armado, la alarma sonará si se abre el cofre y / o el maletero.

6. Modo de Carjack

    Si su vehículo está secuestrado, active esta función y la alarma sonará 60 segundos después de que sea robada.

7. Modo Valet

    Cuando está habilitado, los controles remotos bloquearán y desbloquearán las puertas sin armar la alarma.


Puerta cerradura y Controll:

1. Control frontal cerradura

    Al bloquear o desbloquear las puertas delanteras, todas las otras puertas se bloquearán y desbloquearán.

    Esta característica se puede habilitar solo para una, ambas o ninguna de las puertas frontales.

2. Automático cerradura / Desbloquear

    Cuando esté habilitada, las puertas se bloquearán y desbloquearán automáticamente cuando se encienda y apague el vehículo.

3. Armado automático de última puerta

    Cuando está habilitado, después de que se apaga el motor, la alarma se armará automáticamente 30 segundos después de salir del vehículo.

4. 2x salidas auxiliares

    Estas salidas auxiliares se pueden usar para abrir el maletero, enrollar las ventanas, etc. (se necesitan módulos adicionales).

Seguridad avanzada:

1. Shock Sensor Bypass

    Cuando el control remoto lo omite, la alarma ignorará el sensor de impacto cuando el sistema esté armado.

2. Desarmado de alarma de emergencia (por secuencia de clave)

    Cuando está activado, el sistema de alarma puede desarmarse con una secuencia de encendido "encendido / apagado".

3. Restablecimiento automático de la sirena (30 segundos o 60 segundos)

    La sirena de la alarma se apagará automáticamente después del intervalo establecido a menos que la alarma vuelva a activarse nuevamente.


Remoto:

1. cerradura / Arm

    Motor Off: cerradura Puertas y brazo Motor Encendido: cerradura Solo puertas.

2. Desbloquear / Desarmar

    Motor Apagado: Desbloquear puertas y desarmar Motor Encendido: Desbloquear puertas solamente.

3. Botón de pánico

    Presione el botón de pánico y la sirena sonará durante 60 segundos.

4. Botón auxiliar (Controles Aux 1 y Aux 2)

    Presione el botón auxiliar durante 3 segundos para controlar Aux1, o presione el botón auxiliar una vez y luego bloquee una vez para Aux2.

5. Coche Buscador

    Cuando se active, la alarma parpadeará para ayudarlo a ubicar su vehículo.

6. Modo de Carjack

    Cuando el motor está encendido, presione los botones de bloqueo y desbloqueo para activar el modo de bloqueo automático.

7. Armado y desarmado silenciosos

    Cuando el motor está apagado, presione los botones de bloqueo y desbloqueo para armar o desarmar la alarma sin que la sirena de confirmación emita un pitido.
