Descripción:
Esta subasta cuenta con este relé intermitente electrónico LED de 2 pines, funciona para la mayoría de motocicletas/bicicletas para reparar las bombillas LED relacionadas con la luz de señal de giro hiperintermitente, 
No funciona correctamente los problemas.
Al usar esta luz intermitente LED de 2 pines, puede instalar las bombillas LED para la motocicleta / bicicleta para 
encienda las luces de señal sin tener que cortar los cables para instalar las resistencias de carga y eliminar por completo los problemas de parpadeo rápido del LED.
Además, esta luz intermitente electrónica LED de 2 pines también es ajustable en velocidad. Puede utilizar un destornillador de cabeza plana para
ajuste la perilla en la parte inferior para configurar sus velocidades intermitentes de 50c/m a 200c/m.
Diseño completamente impermeable
Reemplace directamente el intermitente OEM de dos pines
Elimine la necesidad de tocar cables para instalar las resistencias de carga
Permita que las bombillas LED parpadeen a la velocidad normal
Funciona para la mayoría de motocicletas como Honda, Kawasaki, KTM, Suzuki, Yamaha y muchas otras marcas.
