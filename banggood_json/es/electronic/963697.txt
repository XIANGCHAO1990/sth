Características:

Al abrir el diseño esquemático original, la descarga gratuita de la interfaz del software de desarrollo, también permite que sus necesidades sean modificadas, fáciles y convenientes para descargar el programa.
Puede simplemente utilizar sensores, todo tipo de conexión de componentes electrónicos (Como luces LED, zumbador, botones y resistencia fotosensible, etc.), hacer todo tipo de trabajos interesantes.
Con el uso del controlador de microprocesador de alta velocidad (ATMEGA328), la interfaz de desarrollo y el entorno son muy simples, fáciles de entender y muy adecuados para que los principiantes aprendan.

Descripción:

E / S digital: 0 - 13
E / S analógica: 0 - 5
Soporta la función de descarga de ISP
Voltaje de entrada: DC 5V ~ 9V (Sin fuente de alimentación externa cuando está conectado a la computadora)
Voltaje de salida: DC 5V / 3.3V
Utilizando el controlador de microprocesador Atmex Atmega328

Cómo utilizar:

1. Descargue el IDE arduino
Enlace: http: http://arduino.cc/en/Main/Software (Copiar para abrir)
2. Descargue el controlador de chip USB
Enlazar: http://www.5v.ru/zip/ch341ser.zip (Copiar para abrir)
3. Conecte la placa de desarrollo UNO, el controlador se instalará automáticamente
4. Seleccione la UNO del dado
5. Seleccione el puerto COM
6. La mejor opción primero, arduino vienen con procedimientos rutinarios, se queman en él.
