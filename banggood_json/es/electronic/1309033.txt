Descripción:

Este sensor capacitivo de humedad del suelo se distingue de la mayoría de los sensores resistivos en el mercado y utiliza detección capacitiva para detectar la humedad del suelo. Se evita el problema de que el sensor de resistencia se corroe fácilmente y su vida útil se amplía considerablemente.

El sensor tiene un chip de regulador de voltaje incorporado que admite un entorno de trabajo de 3.3 voltios, lo que significa que funciona incluso en una placa de control Arduino de 3.3V. Una PC miniatura como la Raspberry Pi solo necesita un módulo externo de conversión ADC (señal analógica a digital) para funcionar.
¡Con una pantalla externa y una placa base, puedes hablar con tus plantas! Vea si tiene sed y no necesita más agua para humedecer.
Voltaje de funcionamiento: 3.3 VDC
Voltaje de salida: 0 ~ 3.0 VDC
Interfaz: PH2.54-3P
Tamaño: 98 x 23 mm (LxW)
