[Módulo inalámbrico ESP32-CAM]

		Descripción:




			1. ESP32-CAM es una placa de desarrollo de modo dual WIFI + Bluetooth que utiliza antenas y núcleos integrados en PCB basados en chips ESP32. Puede funcionar de forma independiente como sistema mínimo.

			2. ESP integra WiFi, Bluetooth tradicional y BLE Beacon, con 2 CPU LX6 de 32 bits de alto rendimiento, arquitectura de tubería de 7 etapas, rango de ajuste de frecuencia principal de 80MHz a 240MHz, sensor en chip, sensor Hall, sensor de temperatura, etc.

			3. Totalmente compatible con los estándares WiFi 802.11b / g / n / e / iy Bluetooth 4.2, se puede utilizar como modo maestro para construir un controlador de red independiente o como esclavo de otros MCU host para agregar capacidades de red a los existentes. dispositivos.

			4. ESP32-CAM se puede utilizar ampliamente en varias aplicaciones de IoT. Es adecuado para dispositivos inteligentes domésticos, control inalámbrico industrial, monitoreo inalámbrico, identificación inalámbrica QR, señales del sistema de posicionamiento inalámbrico y otras aplicaciones de IoT. Es una solución ideal para aplicaciones de IoT.

- Módulo SoC ultrapequeño 802.11b / g / n Wifi + BT / BLE

- Admite la red de distribución Smart Config / AirKiss One-click

- Soporte para actualización local en serie y actualización remota de firmware (FOTA)
	- Apoyar el desarrollo secundario

[Módulo FT232RL]



		El adaptador serial USB FTDI viene con el chip FT232RL y usa voltaje de operación de 3.3V y 5V. El nivel de lógica del adaptador se puede configurar mediante puentes en la placa, lo que facilita la conexión a su controlador. La alimentación USB tiene protección contra sobrecorriente, utilizando un fusible de auto-restauración de 500MA. Puede utilizar los módulos de este kit para grabar el programa directamente sin compra adicional.



[Hembra a Hembra Dupont Jumper Cables]



		Los cables Jumper están hechos de material de alambre de cobre de alta calidad para un uso seguro y duradero. Este conjunto contiene cables de 20 cm 40 pines macho a macho Jumper, con los que puede completar el proyecto más fácilmente.
