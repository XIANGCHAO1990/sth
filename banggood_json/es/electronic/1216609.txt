Principales instrucciones de operación:



"Prev / V--" tecla: "presione brevemente" para cambiar la canción "anterior", "prensa larga" para cambiar "volumen abajo"

"Next / v ++" tecla: "presione brevemente" para cambiar "canción siguiente", "prensa larga" para cambiar "aumento de volumen"

"P / P / Mode" clave: "presione brevemente" a "play / pause" interruptor, "prensa larga" para el disco U, TF tarjeta "modo" seleccione

Tecla "Repetir": "pulsar brevemente" para el conmutador de ciclo de "canción única / completa" (sin pulsación prolongada)
