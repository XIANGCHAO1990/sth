In addition to all the features of the previous board, the Uno R3 now uses an ATmega16U2 instead of the ATmega8U2 chip. This allows for faster transfer rates, no drivers needed for Linux or Mac (inf file for Windows is needed), and the ability to have the Uno show up as a keyboard, mouse, joystick, etc.  注意：     TOSduino Uno R3は、一部のコンピューターに正しくインストールするために1.0ドライバーフォルダーを必要とします。 R3を古いバージョンのIDEでプログラムできることをテストし、確認しました。ただし、新しいコンピューターでR3を初めて使用するときは、そのコンピューターに1.0をインストールする必要があります。 小さなチップはATmega16U2で、28ピンDIPソケットチップはATmega328-puです。  パフォーマンスの説明：    Digital I / O digital input / output terminal: 0 to 13.   アナログI / Oアナログ入力/出力：0〜5。   Support ISP download function.   入力電圧：コンピューターのUSBに接続した場合、外部電源はありません。外部電源5V〜9VDC電圧入力。   Output voltage: 5V / 3.3V DC.
