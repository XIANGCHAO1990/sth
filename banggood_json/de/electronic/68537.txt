Beschreibung:



UNO R3 Board ist 100% kompatibel mit original Arduino, jetzt hat es perfekte Qualität und Leistung.


Arduino ist eine physikalische Open-Source-Plattform, die auf einer einfachen I / O-Platine basiert und eine Entwicklungsumgebung implementiert, die die Processing / Wiring-Sprache implementiert. Arduino kann verwendet werden, um eigenständige interaktive Objekte zu entwickeln oder mit Software auf Ihrem Computer (zB Flash, Processing, MaxMSP) zu verbinden. Die Open-Source-IDE kann kostenlos heruntergeladen werden (derzeit für Mac OS X, Windows und Linux).
Dies ist der neue Tosduino Uno R3. Zusätzlich zu allen Funktionen des vorherigen Boards verwendet das Uno R3 jetzt einen ATmega16U2 anstelle des ATmega8U2-Chips. Dies ermöglicht schnellere Übertragungsraten, keine Treiber für Linux oder Mac (Inf-Datei für Windows erforderlich), und die Fähigkeit, die Uno erscheinen als Tastatur, Maus, Joystick, etc.

HINWEIS: 

Die TOSduino Uno R3 erfordert die Arduino 1.0 Treiber Ordner, um ordnungsgemäß auf einigen Computern installieren. Wir haben getestet und bestätigt, dass das R3 in älteren Versionen der IDE programmiert werden kann. Jedoch, das erste Mal mit dem R3 auf einem neuen Computer, müssen Sie Arduino 1.0 auf dieser Maschine installiert haben.
Der kleine Chip ist ATmega16U2, der 28-polige DIP-Buchse Chip ist ATmega328-pu.

Leistungsbeschreibung:

Digitaler E / A-Digital-Eingang / Ausgang: 0 bis 13.
Analoge Ein- / Ausgänge Analoge Ein- / Ausgänge: 0 bis 5.
Unterstützung der ISP-Download-Funktion.
Eingangsspannung: Keine externe Stromversorgung, wenn an den Computer angeschlossen USB, externe Stromversorgung 5V ~ 9V DC Spannungseingang.
Ausgangsspannung: 5V / 3.3V DC.
Atmel Atmega328 Mikroprozessor Regler verwenden.
