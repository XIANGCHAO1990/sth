Beschreibung:


UNO R3 Board ist 100% kompatibel mit Original, jetzt hat es perfekte Qualität und Leistung.

ist eine Open-Source-Plattform für physikalische Computer, die auf einer einfachen E / A-Karte und einer Entwicklungsumgebung basiert, die die Programmiersprache Processing / Wiring implementiert. kann verwendet werden, um eigenständige interaktive Objekte zu entwickeln, oder kann mit Software auf Ihrem Computer verbunden werden (z. B. Flash, Processing, MaxMSP). Die Open-Source-IDE kann kostenlos heruntergeladen werden (derzeit für Mac OS X, Windows und Linux).
			Das ist der neue Tosduino Uno R3. Zusätzlich zu allen Features des vorherigen Boards verwendet der Uno R3 nun einen ATmega16U2 anstelle des ATmega8U2-Chips. Dies ermöglicht schnellere Übertragungsraten, keine Treiber für Linux oder Mac benötigt (Inf-Datei für Windows wird benötigt) und die Möglichkeit, die Uno als Tastatur, Maus, Joystick, etc. zu sehen.

Hinweis: 

			Für das TOSduino Uno R3 ist der Treiberordner 1.0 erforderlich, damit es auf einigen Computern ordnungsgemäß installiert werden kann. Wir haben getestet und bestätigt, dass der R3 in älteren IDE-Versionen programmiert werden kann. Bei der ersten Verwendung des R3 auf einem neuen Computer muss jedoch 1.0 auf diesem Computer installiert sein.
Der kleine Chip ist ATmega16U2, der 28-polige DIP-Socket-Chip ist ATmega328-pu.

Leistungsbeschreibung:

			Digitale E / A-Digitaleingang / Ausgangsklemme: 0 bis 13.
			Analog I / O Analoge Ein- / Ausgänge: 0 bis 5.
			Unterstützung ISP Download-Funktion.
			Eingangsspannung: Keine externe Stromversorgung bei Anschluss an den Computer USB, externe Stromversorgung 5V ~ 9V DC Spannungseingang.
			Ausgangsspannung: 5V / 3.3V DC.
			Verwendung des Mikroprozessor-Controllers Atmega328.
