Beschreibung:

USB-Schnittstellenkamera, unterstützt alle Revisionen des Pi
	Fokussierbereich: 20 mm (MAX)
	Videoauflösung: 640*480
	Größe: 3,8 cm * 1,5 cm * 3 cm, Linie skalierbar, die längste ist 65 cm


So verwenden Sie diese Kamera:


1.Fokus einstellen


Testen Sie diese USB-Kamera mit Ihrem Computer und stellen Sie den Fokus auf die beste Bildschirmauflösung ein.

2. Video aufnehmen

sudo apt-get update

sudo apt-get install ffmpeg

avconv -f video4linux2 -r 7 -s 640x480 -i /dev/video0 video0.avi

3.Video abspielen

omxplayer -o hdmi-video0.avi

4.  Foto machen

Installieren Sie die Software:

sudo apt-get install fswebcam

Bilder aufnehmen:

fswebcam -r 640x480 Bild.jpg
