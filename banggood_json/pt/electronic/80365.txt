IIC I2C TWI SP Porta do Módulo de Interface Serial para 5V Arduino 1602 LCD Exibição

Descrição:

Esta é outra excelente interface serial do IIC / I2C / TWI / SPI. Como os recursos de pin do controlador Arduino são limitados, seu projeto pode não ser capaz de usar um escudo LCD normal depois de conectado com uma certa quantidade de sensores ou cartão SD. No entanto, com este módulo de interface I2C, você poderá realizar a exibição de dados por apenas 2 fios. Se você já possui dispositivos I2C em seu projeto, esse módulo LCD realmente não custa mais recursos. É fantástico para o projeto baseado em Arduino.
Endereço I2C: 0X20 ~ 0X27 (o endereço original é 0X20, você pode mudar você mesmo)
a luz de fundo eo contraste são ajustados por potenciômetro
Venha com 2 interfaces IIC, que podem ser conectadas pela linha Dupont ou pelo cabo dedicado da IIC.
Endereço I2C: 0x27 (Endereço I2C: 0X20 ~ 0X27 (o endereço original é 0X27, você pode alterá-lo você mesmo)
