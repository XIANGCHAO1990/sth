Descrição:

Câmera de interface USB, suporta todas as revisões do Pi
	Alcance de focagem: 20MM (MAX)
	Resolução de vídeo: 640*480
	Tamanho: 3,8 cm * 1,5 cm * 3 cm, linha escalável, o mais longo é 65 cm


Como usar esta câmera:


1. Ajuste o foco


Teste esta câmera USB com seu computador e ajuste o foco para corresponder à melhor resolução de exibição.

2. Grave vídeos

sudo apt-get update

sudo apt-get install ffmpeg

avconv -f video4linux2 -r 7 -s 640x480 -i /dev/video0 video0.avi

3.Reproduzir vídeo

omxplayer -o hdmi video0.avi

4.  Tirar fotos

Instale o software:

sudo apt-get install fswebcam

Capturar imagens:

fswebcam -r 640x480 image.jpg
