Descrição:
Este leilão apresenta este relé de pisca-pisca eletrônico LED de 2 pinos, funciona para a maioria das motocicletas/bicicletas para consertar as lâmpadas LED relacionadas à luz de pisca hiper piscando, 
não funciona corretamente problemas.
Usando este pisca-pisca LED de 2 pinos, você pode instalar as lâmpadas LED da motocicleta / bicicleta para 
acenda as luzes de sinalização sem precisar cortar os fios para instalar os resistores de carga e eliminar completamente os problemas de intermitência rápida do LED.
Além disso, este pisca-pisca eletrônico LED de 2 pinos também é ajustável em velocidade. Você pode usar uma chave de fenda de cabeça chata para
ajuste o botão na parte inferior para definir suas velocidades de piscar de 50c/m a 200c/m.
Design totalmente à prova d'água
Substitua diretamente o pisca-pisca de dois pinos OEM
Elimine a necessidade de fios rosqueados para instalar os resistores de carga
Permitir que as lâmpadas LED pisquem na taxa normal
Trabalhe para a maioria das motocicletas como Honda, Kawasaki, KTM, Suzuki, Yamaha e muitas outras marcas também
