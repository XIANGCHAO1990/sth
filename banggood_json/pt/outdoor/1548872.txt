Não sabe como bombear? Basta seguir estas etapas:

1. Pressione e segure o botão liga / desliga por 3 segundos e aguarde a barra de progresso ser concluída.

2. Pressione brevemente o botão set para selecionar 5 modos de cena oficial inflável.

3. Pressione brevemente o botão U / LED para alternar a unidade de pressão de ar.

(1PSI-6,89KPA 1BAR = 100KPA 1KG / cm² = 98KPA)

4. Pressione brevemente os botões +/- predefinidos para ajustar a pressão predefinida. Pressione e segure para ajustar o valor rapidamente.

5. Conecte a mangueira de ar

6. Pressione rapidamente o botão liga / desliga para iniciar.
