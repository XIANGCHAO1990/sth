Noter: pas pleine puceCaractéristique:
	1. Lire l'identification

Affiche l'identification complète de l'unité de contrôle, par ex. Numéro de pièce, version du logiciel / du matériel, fabricant, etc.

2. Lire les codes d'erreur

Affiche tous les codes d'erreur enregistrés et en attente avec une description complète (par ex. "Pression du rail - pression trop basse").

Le programme prend en charge l'impression de rapports ou la copie dans le presse-papier.

3. Effacer les codes d'erreur

Cette fonction efface tous les codes d'erreur et autres informations de diagnostic enregistrés.

4. Auto-scan (balayage / test complet de la voiture)

Détecte tous les calculateurs (unités de commande électroniques) installés dans la voiture et lit tous les codes de diagnostic.

5. Valeurs mesurées

Le programme affiche les données en direct comme la vitesse du moteur, la tension Batterie, le capteur d'oxygène, la température du liquide de refroidissement, etc. Les valeurs peuvent être affichées sous forme graphique, 9 valeurs en même temps ou liste complète (toutes les valeurs mesurées). La journalisation dans un fichier est également prise en charge, ce qui permet une analyse hors ligne.

6. Test de l'actionneur

Le test de l'actionneur active l'actionneur particulier (par exemple, allumer la pompe à carburant, verrouiller / déverrouiller la roue, verrouiller / déverrouiller les portes, couper le carburant, etc.)
