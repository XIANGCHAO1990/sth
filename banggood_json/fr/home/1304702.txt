La description :

1. Installation facile, insérez simplement la barre dans la terre
2. Il est résistant aux intempéries, ce qui lui permet de fonctionner toute l'année et dans pratiquement toutes les conditions météorologiques
3. Utilisez l'énergie solaire pour recharger les batteries afin qu'elles fonctionnent toujours
4. Le répulsif pour animaux utilise des ondes ultrasonores.
5. Le capteur infrarouge passif (PIR) détecte le mouvement et déclenche le haut-parleur à ultrasons émettant un son que les gens ne peuvent pas entendre mais chasse les parasites extérieurs courants comme les chiens, les chats, les cerfs, les ratons laveurs, les souris, les mouffettes, les écureuils et se déplacent. moyen sûr et humain d'éloigner les animaux indésirables

Utilisation:

1. Connectez la tête du répulsif, les sections de poteau, la pièce de connexion et le piquet de terre.
2. L'unité est maintenant prête pour l'installation. L'emplacement doit permettre un ensoleillement suffisant pour maintenir les batteries complètement chargées.
3. Mettez l'interrupteur d'alimentation en position «ON»
4. Pour de meilleurs résultats, maintenez la tête du répulsif à environ 9-10 pouces au-dessus

Remarque : 

1. N'immergez pas le répulsif dans l'eau
2. Ne couvrez pas le capteur PIR
3. Pour nettoyer le répulsif, utilisez un chiffon doux et humide et un savon doux avec de l'eau. N'utilisez pas de produits chimiques agressifs ou d'agents de nettoyage abrasifs qui pourraient endommager le répulsif
