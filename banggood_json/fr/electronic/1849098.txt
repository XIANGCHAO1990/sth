La description:Ce produit est un commutateur sans fil prenant en charge les modes de travail momentané (pas à pas) / autobloquant / verrouillage. Les utilisateurs peuvent ajouter l'appareil à l'APP eWeLink afin de contrôler à distance les appareils ménagers connectés ou les circuits d'alimentation.Remarque: les relais sont des commutateurs sans sortie, l'entrée d'alimentation (AC / DC) uniquement pour le PCBA.Appuyez sur le bouton de réglage du mode de travail manuellement, le produit peut changer le mode de travail. En mode pas à pas, vous pouvez sélectionner deux modes de câblage: -Activer 1s puis arrêt automatique -Éteindre 1s puis allumer automatiquement.En mode autobloquant, vous pouvez allumer / éteindre à distance immédiatement les appareils connectés.En mode de verrouillage (uniquement les supports 2 et 4 groupes), vous pouvez contrôler à distance les moteurs AC / DC.Le produit permet le contrôle à distance des appareils ménagers via l'application mobile eWeLink, disponible sur Android et iOS. Fonctions prises en charge :* Activation / désactivation à distance * Horaires* Partage d'appareils*Gestion de groupe* Compatible avec Amazon Alexa, Amazon Tap* Fonctionne avec Google Home*Fonctionne avec IFTTTCe produit est compatible avec Amazon Alexa. Vous pouvez demander à Alexa de contrôler l'appareil avec les commandes vocales suivantes. Veuillez noter que seules les commandes vocales US / UK anglais et allemand sont désormais prises en charge.Try out these phrases ( {deviceName} should be the name of your device): "Alexa, turn on {deviceName}""Alexa, turn off {deviceName}"«Alexa, schalte Ger?
