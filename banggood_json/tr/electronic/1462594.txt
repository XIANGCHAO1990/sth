Bu kamera nasıl kullanılır:


1. Odağı Ayarla


Bu USB kamerayı bilgisayarınızla test edin ve odağı en iyi ekran çözünürlüğüne uyacak şekilde ayarlayın.

2. Video çek

sudo apt-get güncellemesi

sudo apt-get install ffmpeg

avconv -f video4linux2 -r 7 -s 640x480 -i /dev/video0 video0.avi

3.Videoyu Oynat

omxplayer -o hdmi video0.avi

4.  Fotoğraf çek

Yazılımı yükleyin:

sudo apt-get install fswebcam

Görüntüleri yakalayın:

fswebcam -r 640x480 resim.jpg
