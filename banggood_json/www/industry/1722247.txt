Description:This offline controller is designed for the use of Mini CNC router 1610/2418/3018 and even with more advanced function of tool set, limit switch supportive, Laser supportive; Offline and Bluetooth supportive. Specification:Firmware: GRBL 08, 09, 1.1 and so on Input Voltage: DC 7V- 36VApplicable CNC Router: 3 Axis CNC router 1610; 2418; 3018 Features:1. Support GRBL and benbox firmware. Support CNC 3-axis laser engraving machine.2. The stroke length isn't limited to 30x18 and 24x18, it can be setting other length. Please set the stroke in the command.For example3018, please set the stroke in the command, 30cm length and 18cm width;2418, please set the XY stroke in the command, length 24cm, width 18cm.Function:1: Manual control: manually adjust the position of XYZ three axes as the carving starting point, and manually open or close the spindle.2: File engraving: the offline board can be connected to the grbl control board for engraving after saving the file, and there is no need to connect to the computer.3: Storage files: the offline version is identified as USB mass storage device after connecting to the computer, which can directly access engraving files.Instructions:GRBL Off-line ControllerPress Z+ and Z- , the Z axis will move up and down.Press X+ and X- , the X axis will move left and right.Press Y+ and Y- , the Y axis will move Front and Back.Press SPIN : Spindle on (Laser on ), Spindle off (laser off)Press SMOD: returnPress Z+ for 5 seconds , change the Language to English.SD Card: Included , you need a SD card reader to write into the root directory G-code files .Notice:The Laser Interface is 3 pin on this control board.
