Come utilizzare questa fotocamera:


1. Regola la messa a fuoco


Testare questa fotocamera USB con il computer e regolare la messa a fuoco in modo che corrisponda alla migliore risoluzione del display.

2. Gira video

sudo apt-get update

sudo apt-get install ffmpeg

avconv -f video4linux2 -r 7 -s 640x480 -i /dev/video0 video0.avi

3. Riproduci video

omxplayer -o hdmi video0.avi

4.  Fare foto

Installa il software:

sudo apt-get install fswebcam

Cattura immagini:

fswebcam -r 640x480 image.jpg
