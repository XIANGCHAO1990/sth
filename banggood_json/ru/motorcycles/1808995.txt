Вы можете услышать звук масляного насоса, и загорится значок масляного насоса. Нажмите вниз, чтобы отобразить HOF, выйдите из заправки топливом, значок масляного насоса исчезнет. Процесс откачки воздуха из магистрали должен производиться под наблюдением, когда масло снова попадет в положение маслозаборника машины, его можно остановить. Если в машину попадает слишком много масла, при зажигании будет черный дым.3. Описание переключения режимов контроля температурыОдновременно нажмите кнопку регулировки вверх и кнопку «ОК», чтобы выполнить переключение режима контроля температуры. Ручной контроль температуры (регулировка массы масла показывает P-16, число представляет массу масла), автоматический контроль температуры (регулировка температуры показывает 25 ℃, число представляет температуру). Разница между двумя способами контроля температуры заключается в том, что: в автоматическом режиме насосное масло может достигать максимума при настройке параметров, когда машина сильно нагревается, ручное перекачивание ограничивается текущим заданным значением, которое не будет -ach максимальное значение масла, установленное параметром. Плюс четко показаны шестеренки, комплектуются детали и узлы, очень удобно для старых рук.4. Описание кода соответствия с помощью пульта дистанционного управленияВ выключенном состоянии сначала одновременно нажмите кнопки питания и OK на пульте дистанционного управления, затем появится пульт дистанционного управления, ожидающий соответствующего текста, затем удерживайте долгое нажатие кнопок регулировки вниз на панели LCD. Он показывает HFR - войдите в интерфейс удаленного сопоставления кода, в случае успеха автоматически выйдите из интерфейса, затем он покажет нормальный рабочий параметр. Если совпадение кода не удалось, он не перейдет в нормальное состояние отображения рабочих параметров. Состояние совпадения кода будет автоматически отключено, если тайм-аут кода удаленного управления не получен.Пакет включает в себя:1xКомплект нагревателя （выберите свой тип)
