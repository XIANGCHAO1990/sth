Описание : 

Мини-универсальный гибкий магнитный держатель для подставки, стенд, индикатор теста Инструмент, набор

Это хороший индикатор для вас! ~
Магнитная подставка с переключаемым постоянным магнитом для удобной установки на плоские железные или стальные поверхности.
Рычаг может быть гидравлически заблокирован в любом положении с помощью ручки на стойке.
Полезно для удерживания индикаторов шкалы или индикаторов проверки циферблата во время операций выравнивания.

Особенности : 

Работа с постоянным магнитом с ВКЛ ВЫКЛ контроль.
Основание с V-образным пазом для надежной установки на цилиндрические поверхности.
Идеально подходит для удержания индикаторов рычага в любом положении на станке для операций центровки.
Гибкие, прочные и точные поворотные соединения могут регулировать любое направление и положение.
Мощное магнитное основание с переключателем включения / выключения магнита.
Обладая высокой стабильностью, он станет вашим идеальным инструментом для измерения прецизионных заготовок.

Один магнитный основание.
Его можно использовать для Индикатор циферблата и Индикатор с рычажным циферблатом, и т.д
Разработан с тремя шарнирами.
Легко использовать для изделий из металла.
На базе есть переключатель включения / выключения.
