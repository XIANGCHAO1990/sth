Описание:M5Paper v1.1 - это новейшее базовое устройство M5Stacks с сенсорным дисплеем E-ink. Это наше первое устройство на базе ESP32-D0WDQ6-V3, в котором интегрирован дисплей E-ink большого размера 540 * 960 @ 4,7 дюйма ，, который поддерживает 16 уровней градаций серого. Дисплей представляет собой емкостный сенсорный экран GT911 ，, который поддерживает касание по двум точкам и разнообразные средства управления жестами. По сравнению с обычным ЖК-дисплеем displays дисплеи с электронными чернилами более удобны для глаз, что делает их отличным выбором для чтения или просмотра в течение более продолжительных периодов времени. Другими преимуществами являются низкое энергопотребление и способность сохранять изображение Даже при отключении питания дисплея в CoreInk встроены многофункциональная кнопка для работы, датчик температуры и влажности SHT30, физические кнопки и порт для карты памяти TF (microSD) для хранения данных.Кроме того, внутренняя микросхема eeprom FM24C02 с 2К-битным (256x8) EEPROM может использоваться для хранения важных данных, даже когда устройство выключено. Батарея Lipo емкостью 1150 мАч обеспечивает работу устройства в течение длительного времени, а срок службы батареи можно дополнительно продлить, используя RTC (BM8563), чтобы перевести устройство в режим глубокого сна и снова вывести его из спящего режима при необходимости。 Включены три порта расширения HY2.0-4P которые позволяют создавать сложные проекты с использованием существующих датчиков в экосистеме M5Stack.Предупреждение: Не подвергайте длительному воздействию ультрафиолетовых лучей, иначе это может вызвать необратимое повреждение экрана E-ink. Решение для управления питанием с низким энергопотреблением, принятое в M5Paper, отличается от решений CORE и StickC. При использовании кнопка PWR используется как кнопка включения (длительное нажатие 2 с). Если вам нужно выключить устройство, вам нужно использовать программный API или нажать кнопку сброса на задней панели, при использовании источника питания USB его нельзя выключить.Функции:ESP32 Стандартные беспроводные функции WiFi 、 BluetoothВнутренний 16М FlashНизкая мощность ДисплейМноготочечный сенсорный экранУгол обзора 180 градусовВстроенный МагнитВнутренний 1150 мАч БатареяРасширяемый - HY2.0-4P 3 x внешних порта расширенияПриложения:Терминал Интернета вещейЭлектронная книгаПромышленная панель управленияПанель Умного ДомаВ пакет включено:1x M5Paper v1.1Документ:EasyLoaderPinMapM5Paper-HY2.0 4P портОтображаемые контакты АЦП / ЦАП ESP32Связанная ссылкаПримерСхемавидео
