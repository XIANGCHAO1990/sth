Описание:

Камера с интерфейсом USB, поддерживает все версии Pi
	Диапазон фокусировки: 20 мм (MAX)
	Разрешение видео: 640*480
	Размер: 3,8 см * 1,5 см * 3 см, масштабируемая линия, самая длинная 65 см


Как использовать эту камеру:


1. Отрегулируйте фокус


Протестируйте эту USB-камеру на своем компьютере и отрегулируйте фокус, чтобы он соответствовал наилучшему разрешению экрана.

2. Снять видео

sudo apt-получить обновление

sudo apt-get установить ffmpeg

avconv -f video4linux2 -r 7 -s 640x480 -i /dev/video0 video0.avi

3.Воспроизвести видео

omxplayer -o hdmi video0.avi

4.  Сфотографировать

Установите программное обеспечение:

sudo apt-get установить fswebcam

Захват изображений:

fswebcam -r 640x480 image.jpg
