Инструкция по эксплуатации:

1. Адаптер питания переменного тока в сетевую розетку, другой конец вилки постоянного тока в перо для рисования - это только «отверстие для входа питания». Желтый индикатор LED показывает, что оборудование имеет питание для перехода в режим ожидания сервопривода.
2. На экране LCD отобразятся слова «» или «», выберите «выбор материала, выбор материала». с экрана LCD. Выбор должен быть таким же, как атрибут материала, согласованный, чтобы избежать сбоя.
3. Нажмите кнопку «Провод», загорится красный LED, это означает, что оборудование перешло в состояние прогрева. Примерно через 0,5-2 минуты. Когда красный LED загорится с красного на синий. Покажите конец предварительного нагрева, оборудование может работать нормально.
4. Расходные материалы вставить конец ручки только «отверстие для ввода питания», принесли с собой долгое нажатие кнопки «провод», случайные поставки транспортного оборудования от встроенного двигателя. Когда конец сопла с составным NIB должен быть воспроизведен успешно. * *
5. 4 творения. (Это оборудование с функцией плавного регулирования скорости, в процессе создания, в соответствии со скоростью пера для регулировки количества клея. Без обеих рук.)
6. Однородные материалы имеют разную температуру плавления из-за разного цвета. пожалуйста, согласно шелку, чтобы своевременно настроить температуру. (оценка точки плавления материала и регулировка температуры см. в разделе «настройка температуры»)
