Этот переходник резервуара для смесителя IBC предназначен для установки в большинство стандартных контейнеров IBC для Великобритании и Европы, где опорная нить представляет собой грубую нить S60 x 6. Существуют разные соединения, которые могут соответствовать разным размерам шлангов.

Модель: DN40 / 50
Материал: пластик с уплотнительным кольцом
Черный цвет  
Вход: S60 x 6 Грубая Резьба
Выход: 12/19/25/32/38/50 мм
Нажмите Connect: 1/2 '', 3/4 '', 1 '', 1-1 / 4 '', 1-1 / 2 '', 2 ''
Подходит для: IBC Tank Use
Размер:

# 2: 3/4 '' Tap Connect to 19mm Шланг
# 3: 1 '' Tap Connect to 25mm Шланг
# 4: 1-1 / 4 '' Tap Connect to 32mm Шланг
# 5: 1-1 / 2 '' Tap Connect to 38mm Шланг
# 6: 2 '' Нажмите для подключения к 50 мм Шланг

Особенности:
1. Нетоксичные и экологически чистые, безопасные, долговечные и долгосрочные для использования.
2. На выбор предлагаются разные резьбовые соединения, которые могут быть подключены к 12/19/25/32/38/50 мм садовому шлангу в соответствии с вашими потребностями.
3. Адаптер IBC / грубая резьба для подключения дренажа для прямого подключения к резервуару для воды IBC.
4. Подходит для большинства стандартных контейнеров IBC в Великобритании и Европе, где опорная нить - грубая нить S60X6.
